Models

Interactive playground index

These browser models are meant to support intuition for switching, signaling, pattern formation, and collective biological behavior. The homepage keeps the full tabbed version inside the floating terminal; this standalone page mirrors the same model index in a simpler layout.

This page is currently unlinked from the main public navigation while the site prioritizes core profile, research, projects, notes, teaching, CV, and contact content.

Important: these models are qualitative browser sketches meant to make collective behavior intuitive. They are not replacements for calibrated scientific simulation code.

Toggle lattice preview

The first implemented model is the toggle lattice: a coarse, GFP/RFP-like switching field that emphasizes local coupling, stochasticity, and domain formation.

Other entries remain intentionally marked as planned until their browser implementations are ready.

The model index loads from assets/data/models.json when static file requests are available. If you opened this page from file://, use a simple local server to load the structured model cards.
